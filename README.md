# The Household

A browser-based management game about family, fortune, staff, and reputation in Victorian Britain.

## Prototype

Open `index.html` in a modern browser. No build step or server is required.

The first prototype includes household creation, origin and marriage trade-offs, staff hiring and dismissal, monthly accounts, four player-directed investment classes, social events, local saving, and an open-ended legacy assessment.
